## Instalação

``` bash
composer require codemastersolutions/code-acl
```

## Instruções de uso

### Em breve

### Testes Automatizados

``` bash
composer test
```

ou

``` bash
./phpunit
```

### Testes Automatizados com Observabilidade de Alterações

``` bash
./phpunit-watcher
```

### Filtrando testes

``` bash
./phpunit-watcher --filter=nome-do-teste
```
